Create and manage frontend listings and forms easily. A flexible plugin for building directories, submission forms, and displaying custom post types on your site.

== Description ==

**FaL Forms and Listings** lets you create listing directories and customizable frontend forms with no coding required.

Key features:
- Custom post type “Listings” (fdg_listings)
- Frontend form builder with custom fields
- Display listings via shortcodes or blocks
- Basic layout and style customization
- Drag-and-drop field arrangement (planned)

Perfect for real estate, job boards, classifieds, or any user-submitted content.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fal-forms-and-listings` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Use the “FaL Listings” menu in the WordPress admin to start creating forms and listings.

== Frequently Asked Questions ==

= Can I customize the form fields? =
Yes, you can add, remove, and rearrange fields in the form builder interface.

= Is this compatible with Elementor or other page builders? =
The plugin outputs listings and forms via shortcodes, so it should work with any page builder that supports shortcodes.

= Does it support custom post types? =
Currently, the plugin works with its own post type `fdg_listings`. Support for additional CPTs is planned.
